# 基本技能

本章的学习目的是为了学习在整个学习过程中所需要掌握的基础技术和能力，是每个人都需要掌握的。对于新手来说，我们只要能够基本掌握，让自己可以用下去就可以了，在使用的过程中遇到了问题可以通过搜索引擎去寻找答案，不推荐大家以完美主义者的心态去学习。

学完本章后，你将：

- 学会如何提问
- 学会 markdown 的使用
- 学会如何写 Blog
- 学会 Git 的简单应用
