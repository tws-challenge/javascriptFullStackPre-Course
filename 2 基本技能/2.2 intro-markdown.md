# Markdown 使用

#### 本节学习目标
* 能够熟练掌握 Markdown 语法，并能够使用 Markdown 来写博客。

#### 学习内容
- 各种header, 如`#`, `##`, `###`, 等等，以及`=====`, `------`
- 文字修饰：`**`, `__`
- 分隔符 `------`
- 列表：`-`以及`1.`
- 插入图片：`![]()`
- 插入链接：`[]()` 及 `<>`
- 引用： `>`
- 行内代码：``code``，以及代码块 ```` ````
- 表格
